### 2.3 各传感器详细接线

#### DHT22 温湿度传感器
```
DHT22          ESP32
─────          ─────
VCC    ───→    3.3V
DATA   ───→    GPIO 4
GND    ───→    GND

注意：DATA 线需要 4.7K 上拉电阻到 VCC
```

#### BH1750 光照传感器
```
BH1750         ESP32
─────          ─────
VCC    ───→    3.3V
GND    ───→    GND
SDA    ───→    GPIO 21
SCL    ───→    GPIO 22
ADDR   ───→    GND (地址 0x23)
```

#### 土壤湿度传感器（电容式）
```
土壤湿度       ESP32
──────         ─────
VCC    ───→    3.3V
GND    ───→    GND
AO     ───→    GPIO 36
```

#### pH 传感器 (pH-4502C)
```
pH传感器       ESP32
──────         ─────
VCC    ───→    5V
GND    ───→    GND
PO     ───→    GPIO 39

注意：输出电压范围 0-3.3V，需校准
```

#### MH-Z19B CO₂ 传感器
```
MH-Z19B        ESP32
──────         ─────
VCC    ───→    5V
GND    ───→    GND
TX     ───→    GPIO 16 (RX)
RX     ───→    GPIO 17 (TX)
```

#### 继电器与水泵
```
继电器         ESP32 / 水泵
─────          ──────────────
VCC    ───→    5V
GND    ───→    GND
IN     ───→    GPIO 25
COM    ───→    水泵正极
NO     ───→    12V 电源正极

水泵负极 ───→ 12V 电源负极
```

## 四、开发环境搭建

### 4.1 安装 Arduino IDE

1. 下载：https://www.arduino.cc/en/software
2. 安装后打开 Arduino IDE

### 4.2 添加 ESP32 开发板支持

1. 打开 `文件` → `首选项`
2. 在"附加开发板管理器网址"中添加：
   ```
   https://dl.espressif.com/dl/package_esp32_index.json
   ```
3. 打开 `工具` → `开发板` → `开发板管理器`
4. 搜索 "esp32"，安装 "ESP32 by Espressif Systems"

### 4.3 安装所需库

打开 `工具` → `管理库`，搜索并安装：

| 库名 | 作者 | 用途 |
|------|------|------|
| DHT sensor library | Adafruit | DHT22 温湿度传感器 |
| Adafruit Unified Sensor | Adafruit | DHT 依赖库 |
| BH1750 | Christopher Laws | 光照传感器 |
| PubSubClient | Nick O'Leary | MQTT 客户端 |
| ArduinoJson | Benoit Blanchon | JSON 数据处理 |
| LoRa | Sandeep Mistry | LoRa 通信（可选） |

## 五、代码配置与烧录

### 5.1 配置 WiFi 参数

打开 `sensor_read（传感器采集）.ino`，修改以下内容：

```cpp
// ==================== WiFi配置 ====================
const char* ssid = "YourWiFiSSID";         // 改为你的 WiFi 名称
const char* password = "YourWiFiPassword"; // 改为你的 WiFi 密码
```

### 5.2 配置 MQTT 参数

```cpp
// ==================== MQTT配置 ====================
const char* mqtt_server = "zhiyun-iot.com"; // MQTT 服务器地址
const int mqtt_port = 1883;                 // MQTT 端口
const char* mqtt_user = "device_01";        // 改为你的设备 ID
const char* mqtt_password = "password";     // 改为你的设备密码
```

### 5.3 烧录步骤

1. 用 USB 线连接 ESP32 到电脑
2. 打开 `工具` → `开发板` → 选择 "ESP32 Dev Module"
3. 选择正确的端口（如 COM3）
4. 点击"上传"按钮（→）
5. 等待烧录完成

### 5.4 查看串口数据

1. 烧录完成后，打开 `工具` → `串口监视器`
2. 设置波特率为 115200
3. 应该看到类似输出：

```

System initialized!
--- Sensor Data ---
Temperature: 25.3 ℃
Humidity: 65.2 %RH
Soil Moisture: 45.0 %
Light: 1250 lux
pH: 6.8
CO2: 520 ppm
Alert: NO
-------------------
```

## 七、系统测试

### 7.1 单传感器测试

建议先单独测试每个传感器，确保接线正确：

```cpp
// DHT22 测试代码
#include <DHT.h>
#define DHTPIN 4
#define DHTTYPE DHT22

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(115200);
  dht.begin();
}

void loop() {
  float t = dht.readTemperature();
  float h = dht.readHumidity();
  Serial.print("Temp: "); Serial.print(t);
  Serial.print(" ℃, Humi: "); Serial.print(h);
  Serial.println(" %");
  delay(2000);
}
```

### 7.2 完整系统测试

1. **串口监视器测试**：查看所有传感器数据是否正常
2. **MQTT 测试**：使用 MQTT 客户端工具订阅主题查看数据
3. **远程控制测试**：发送 `pump_on` / `pump_off` 指令测试水泵
4. **预警测试**：改变环境条件（如用吹风机加热）测试预警功能

### 7.3 常见问题排查

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| WiFi 连接失败 | SSID/密码错误 | 检查 WiFi 配置 |
| DHT22 读数 NaN | 接线错误或上拉电阻缺失 | 检查接线，添加 4.7K 上拉电阻 |
| 光照值为 0 | I2C 地址错误 | 检查 ADDR 引脚连接 |
| pH 值不准 | 未校准 | 使用标准缓冲液校准 |
| CO₂ 读数 -1 | 波特率不匹配 | 确认 MH-Z19B 使用 9600 波特率 |
| MQTT 连接失败 | 服务器地址错误 | 检查 MQTT 配置 |

## 九、部署建议

### 9.1 室内部署

```
┌─────────────────────────────────────────────┐
│                  微农园                      │
│                                             │
│    [传感器1]    [传感器2]    [传感器3]       │
│        ↓            ↓            ↓          │
│    ┌─────────────────────────────────┐      │
│    │         网关（树莓派）           │      │
│    └─────────────────────────────────┘      │
│                    ↓                        │
│              WiFi / 4G                      │
│                    ↓                        │
│              云平台服务器                    │
└─────────────────────────────────────────────┘
```

### 9.2 户外部署注意事项

1. **防水**：使用防水接线盒
2. **供电**：太阳能 + 蓄电池方案
3. **固定**：使用扎带或支架固定传感器
4. **防晒**：避免阳光直射传感器
5. **防雷**：接地保护
